Pour ajouter les données, Vous pouvez modifier dans les fichiers json et exécutez les instruction suivants pour mise à jour la base de données.


Pour les utilisateur linux :
- exécutez le fichier "creationBase.sh" pour charger la base de données

Pour les utilisateur Windows :
- Ouvrez le fichier "mongoimport.bat"
- Modifiez le path vers "mongoimport" de MONGODB installé dans votre windows
- Enregistrez le fichier
- Exécutez le fichier "mongoimport.bat" pour charger la base de données
